DOM Manipulation

Element Description
document.getElementById()Selects an element by its ID.
document.querySelector()Selects the first element that matches a CSS selector.
document.querySelectorAll()Selects all elements that match a CSS selector.
document.createElement()Creates a new HTML element.
element.appendChild()Appends a node as the last child of an element.
element.removeChild()Removes a child node from an element.
element.setAttribute()Sets the value of an attribute on the specified element.
element.getAttribute()Gets the value of an attribute on the specified element.
element.classList.add()Adds one or more class names to an element.
element.classList.remove()Removes one or more class names from an element.
element.classList.toggle()Toggles a class name on or off for an element.

Event Handling

Element Description
addEventListener()Attaches an event handler to an element.
removeEventListener()Removes an event handler from an element.
event.preventDefault()Prevents the default action of the event.
event.stopPropagation()Stops the event from propagating up the DOM tree.
onclickEvent that fires when an element is clicked.
onmouseoverEvent that fires when the mouse is moved over an element.
onmouseoutEvent that fires when the mouse is moved out of an element.
onkeydownEvent that fires when a key is pressed down.
onkeyupEvent that fires when a key is released.
onsubmitEvent that fires when a form is submitted.

Functions and Control Structures

Element Description
functionDefines a function.
returnExits a function and returns a value.
if/elseDefines a conditional statement.
forCreates a loop that is executed as long as a condition is true.
whileCreates a loop that is executed as long as a condition is true.
do/whileCreates a loop that is executed at least once, and then continues to loop as long as a condition is true.
switchEvaluates an expression, matching the expression's value to a case clause, and executes statements associated with that case.
try/catchMarks a block of statements to try, and specifies a response, should an exception be thrown.
async/awaitUsed to handle asynchronous operations.

Arrays and Looping

Element Description
ArrayCreates a new array.
array.push()Adds one or more elements to the end of an array.
array.pop()Removes the last element from an array and returns that element.
array.shift()Removes the first element from an array and returns that element.
array.unshift()Adds one or more elements to the beginning of an array.
array.map()Creates a new array with the results of calling a function on every element in the array.
array.filter()Creates a new array with all elements that pass the test implemented by the provided function.
array.reduce()Applies a function against an accumulator and each element in the array to reduce it to a single value.
array.forEach()Executes a provided function once for each array element.
array.find()Returns the value of the first element in the array that satisfies the provided testing function.

Objects and JSON

Element Description
ObjectCreates a new object.
object.keyAccesses a property of an object.
object.method()Calls a method on an object.
JSON.stringify()Converts a JavaScript object or value to a JSON string.
JSON.parse()Parses a JSON string, constructing the JavaScript value or object described by the string.
Object.keys()Returns an array of a given object's own enumerable property names.
Object.values()Returns an array of a given object's own enumerable property values.
Object.entries()Returns an array of a given object's own enumerable property [key, value] pairs.
object.hasOwnProperty()Returns a boolean indicating whether the object has the specified property as its own property.
classDefines a class in JavaScript, which is a blueprint for creating objects.

fetch()Used to make network requests to servers.
MathProvides properties and methods for mathematical constants and functions.
